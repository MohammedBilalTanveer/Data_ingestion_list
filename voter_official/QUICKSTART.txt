## рҹ”§ Configuration

### Default Settings
```python
# backend_api.py
DISTRICT = "BANGALORE URBAN"
AC_NUMBER = 88              # Yelahanka
MAX_PARTS = 527

OUTPUT_DIR = "./output"
PDF_DIR = "./output/pdfs"

PORT = 5000                 # Flask
TIMEOUT = 60                # Download timeout in seconds
MAX_RETRIES = 3             # Download retry attempts
```

### Change Configuration

**Custom District/AC**:
Edit `backend_api.py`:
```python
DISTRICT = "YOUR_DISTRICT"
AC_NUMBER = 150
```

**Custom Port**:
Edit `backend_api.py`:
```python
app.run(port=8000)
```

**Custom Download Folder**:
Edit `backend_api.py`:
```python
OUTPUT_DIR = "D:/MyVoterData"
```

## рҹҺ“ Usage Examples

### Download PDFs
1. Open http://localhost:5173
2. Click "рҹ“Ҙ Download PDFs" tab
3. Choose mode:
   - Range: 1-10 (for first 10 PDFs)
   - Range: 278-278 (for only PDF 278)
   - Specific: 1, 278, 300 (comma-separated)
4. Click "в¬ҮпёҸ Start Download"
5. Wait for completion (shows progress bar)

### Search PDFs
1. Click "рҹ”Қ Search" tab
2. Check PDFs to search (or "Select All")
3. Type search query (min 2 characters)
4. Click "рҹ”Қ Search"
5. View results with:
   - Match number
   - Page number
   - Context (previous/next lines)
6. Click "Load more" to see more results

### Example Searches
- Search: "аӨ°аӨҫаӨ№аҘҒаӨІ" (Kannada name)
- Search: "278" (number/reference)
- Search: "Yelahanka" (location)
- Search: "аӨҶаӨҰаӨ°" (any Kannada text)
